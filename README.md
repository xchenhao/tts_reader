# TTS Reader

## Feature
- 基于 Flutter 的文本 TTS 朗读器
- 支持 Windows、macOS、Android
- TTS 源：OpenAI TTS、Microsoft TTS
- 支持设置代理
- 支持设置应用主题
- 支持倍速
- 支持书签
- 支持点击指定地方朗读
- 支持 txt、epub、贴粘文本
- 支持从上次地方开始朗读
